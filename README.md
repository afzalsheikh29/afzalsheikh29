## About Me

I am a Cloud and DevOps Engineer with hands-on experience in designing, automating, and maintaining secure cloud infrastructure.

My work focuses on CI/CD pipeline automation, containerized deployments, infrastructure as code, and AWS-native production systems. I build solutions that improve deployment speed, system reliability, and operational efficiency.

## Core Expertise

- Cloud infrastructure engineering (AWS primary)
- CI/CD pipeline design and automation
- Docker and Kubernetes orchestration
- Infrastructure as Code using Terraform and CloudFormation
- Serverless and event-driven architectures
- Monitoring, logging, and observability
- Cloud security and cost optimization
