## About Me

I am a Cloud and DevOps Engineer focused on building secure, scalable, and automated cloud infrastructures.  
I specialize in CI/CD pipelines, container orchestration, infrastructure as code, and AWS-based production systems.

I work with cloud-native tools to automate deployments, improve system reliability, and reduce operational overhead.
